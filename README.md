# 크랙 장기 요약 메모리 편집기
- 크랙 장기 요약 메모리를 텍스트 에디터 형식으로 한번에 통합 편집할 수 있도록 하는 확장입니다.
- 이 확장은 gemini를 이용하여 제작되었습니다.
- 다운로드 : https://github.com/hamster4762/crack-summary-editor/raw/refs/heads/main/crack-summary-editor.user.js
